# 第3周 Day1：DAS 基线实验报告

## 1. 实验目标

本实验实现 DAS（延迟求和波束形成）基线方法。
给定声源位置和麦克风坐标，先计算声源到各麦克风的近场传播延迟，
再将多通道信号按延迟对齐并叠加平均，得到增强后的目标声音。

本实验分别使用两类位置：

- 真实声源位置：用于验证 DAS 代码本身是否正确；
- 第2周 SRP-PHAT 估计位置：用于验证定位结果是否能用于后续声音增强。

评价时，单通道输出与 clean mic0 比较；DAS 输出与同一位置下的 clean DAS 参考比较，
避免把多通道对齐叠加后的 DAS 输出错误地拿去和单个麦克风 clean 信号比较。

## 2. 实验配置

- 场景数：100
- 麦克风布局：`mic_8`
- 采样率：48000 Hz
- RT60：0.30 s
- SNR：20.0 dB
- 声源模式：`probe`
- 搜索网格间距：0.02 m

## 4. 验收判断

- 真实位置 DAS 平均 SI-SDRi = 7.21 dB，大于 0 dB，说明真实位置 DAS 优于单通道。
- 估计位置 DAS 平均 SI-SDRi = 7.07 dB，大于 0 dB，说明第2周定位结果可以用于 DAS 增强。
